DIFFERENCE BETWEEN Div and Span Tags
* Div : is a block level element.
* Span : is an inline element.

* Div : can be used to group block level elements.
* Span : can be used to group inline elements.

* Div : can be used to create a section in a document.
* Span : can be used to create a section in a paragraph.

* Div : can be used to create a container for a section.
* Span : can be used to create a container for a section in a paragraph.

* Div : Manipulates HTML directly
* Span : Manipulates CSS directly
